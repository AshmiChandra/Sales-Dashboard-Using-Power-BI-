# Sales-Dashboard-Using-Power-BI-

## Overview
This project is a simple interactive Sales Dashboard created using Microsoft Power BI.  
The dashboard provides insights into sales performance, customer distribution, discounts, regions, and payment methods through different visualizations.

## Features
- KPI Cards displaying:
  - Total Product Count
  - Total Sales Amount
  - Average Discount

- Visualizations Included:
  - Region-wise Sales Contribution (Pie Chart)
  - Sales Trend by Date (Bar Chart)
  - Average Sales Amount by Unit Price (Scatter Plot)
  - Sales Amount by Discount (Line Chart)
  - Sales Representative Contribution
  - Customer Type Distribution

- Interactive Filters/Slicers:
  - Region
  - Product Category
  - Payment Method

## Tools & Technologies

The dashboard helps in:
